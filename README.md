# 🧠 ИИ-Психолог

AI-ассистент психологической поддержки на Android. Ведёт текстовый и голосовой
диалог на русском языке, работает в России без VPN и автоматически распознаёт
кризисные сообщения, направляя пользователя к живому специалисту.

---

## ✨ Возможности

- **Чат с ИИ-психологом** — тёплый, поддерживающий стиль, активное слушание
- **Российская нейросеть GigaChat** (Сбербанк) — работает без блокировок и VPN
- **Кризис-детектор** — 2 уровня (высокий стресс / кризис), 40+ ключевых фраз.
  При кризисе сообщение в ИИ не отправляется, а сразу показывается диалог
  с кнопкой звонка на телефон доверия **8-800-2000-122**
- **Голосовой ввод/вывод** — кнопка микрофона в чате (Speech-to-Text → GigaChat → Text-to-Speech)
- **Режим «телефонного звонка»** — непрерывный цикл *слушает → думает → говорит*
  с пульсирующей анимацией
- **Светлая / тёмная тема** — переключение в настройках, сохраняется между запусками
- **Отчёты об ошибках** — автоматическая отправка HTML-письма с историей диалога
  и характеристиками устройства через SMTP

## ⚙️ Настройка перед сборкой

> ⚠️ Секретные ключи вынесены в `Secrets.java`, который **не хранится в git**.
> Без него проект не соберётся.

### Шаг 1. Создайте `Secrets.java`

Скопируйте шаблон и переименуйте:

```
app/src/main/java/com/example/aipsy/Secrets.example.java
              ↓ копировать и переименовать ↓
app/src/main/java/com/example/aipsy/Secrets.java
```

### Шаг 2. Впишите свои значения

```java
public final class Secrets {
    // developers.sber.ru → вкладка "Ключи" → "Авторизационные данные"
    public static final String GIGACHAT_AUTH_KEY = "ВАШ_КЛЮЧ_GIGACHAT";

    // SMTP-аккаунт для отправки отчётов (опционально)
    public static final String FROM_EMAIL   = "отправитель@yandex.ru";
    public static final String APP_PASSWORD = "пароль_приложения"; // Яндекс ID → Пароли приложений
    public static final String TO_EMAIL     = "получатель@gmail.com";

    private Secrets() {}
}
```

**Где взять ключ GigaChat:** зарегистрируйтесь на
[developers.sber.ru](https://developers.sber.ru/portal/products/gigachat),
создайте проект (тариф **Freemium** — бесплатно) и скопируйте
«Авторизационные данные» (готовый base64-ключ).

## 🏗 Сборка

1. Откройте проект в **Android Studio**
2. Создайте `Secrets.java` (см. выше)
3. `File → Sync Project with Gradle Files`
4. `Build → Rebuild Project`
5. Запустите на устройстве или эмуляторе (Android 11+)
